## Building

1. Fork [`clipper`](https://github.com/flazepe/clipper) under the same parent directory as `clipper-gui`.
2. Make sure to have `tauri-cli` installed (`cargo install tauri-cli --locked`).
3. Run `cargo tauri build`.
